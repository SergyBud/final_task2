Ресурс: https://www.labirint.ru/

Тестовые сценарии проверяют по порядку:
AughPage - проверка авторизации на сайте
NegativeAuthPage - ввод невалидных значений при авторизации пользователя
LabirintPagee - проверка функционала поиска по сайту
ADDbookPage - проверка функционала личного кабинета, вкладок в личном кабинете
              Редактирование отдела настроек
              Переход по ссылкам на в верхней панели сайта

Для запуска необходимо написать в терминале: pytest tests.py

Тесты необходимо выболнять при открытом браузере на весь экран.
